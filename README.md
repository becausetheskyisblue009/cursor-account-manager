# Cursor Account Manager

A secure desktop application for managing multiple Cursor IDE accounts.

## What It Does

Cursor Account Manager helps you organize and manage multiple Cursor IDE accounts in one place. Store account credentials, switch between accounts easily, and keep track of account status - all with your data stored locally and securely.

## Key Features

- **Multiple Account Management** - Store and organize unlimited Cursor IDE accounts
- **Secure Local Storage** - All data encrypted and stored locally on your device
- **Quick Search & Sort** - Find accounts instantly with smart search
- **Smart Masking** - Secure credential display with one-click copy
- **Status Tracking** - Monitor account status (Active, Inactive, Expired, etc.)
- **Light & Dark Themes** - Choose your preferred theme
